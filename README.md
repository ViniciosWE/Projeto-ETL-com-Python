# Projeto ETL com Python

## Sobre o Projeto

Este projeto implementa um pipeline de dados utilizando o processo ETL (Extract, Transform, Load). O objetivo é simular um cenário real onde dados de clientes são processados para gerar informações úteis, como mensagens personalizadas e classificação de perfil.

A aplicação foi desenvolvida utilizando Python e a biblioteca Pandas, com foco em organização de código e boas práticas.

---

## Objetivo

- Demonstrar o funcionamento de um pipeline ETL
- Trabalhar com leitura e escrita de arquivos CSV
- Aplicar regras de negócio em dados
- Estruturar o código de forma modular

---

## Tecnologias Utilizadas

- Python 3
- Pandas

---

## Estrutura do Projeto

O projeto está organizado da seguinte forma:
Desafio/
│
├── data/
│ ├── entrada.csv
│ └── saida.csv
│
├── src/
│ ├── extract.py
│ ├── transform.py
│ └── load.py
│
└── main.py


---

## Como Executar

1. Instalar as dependências:
***pip install pandas***


2. Executar o projeto:
***python main.py***
